# 렌탈어때 웹서비스 요약

## 서비스 개요
- 렌탈어때는 기업/기관 대상 렌탈 중심 B2B 쇼핑몰입니다.
- 사용자 회원가입/로그인, 상품 탐색, 예약 신청, 관리자 운영 기능을 제공합니다.

## 주요 기능
- 회원 기능: 회원가입, 로그인, 마이페이지, 1:1 문의, 알림 확인
- 상품 기능: 카테고리 기반 상품 조회, 상세 확인, 예약 요청
- 관리자 기능: 상품/카테고리/섹션/CMS/회원/문의 관리

## 기술 구성
- 프론트엔드: React + Vite + TypeScript
- 인증: Firebase Authentication
- 데이터/스토리지: Supabase(PostgreSQL, Storage)
- 이메일 인증: Firebase Functions 기반 발송

## 운영 포인트
- Firebase는 인증 중심, Supabase는 서비스 데이터 중심으로 분리해 운영합니다.
- Supabase 신규 프로젝트 연결 시 초기 스키마 적용이 선행되어야 정상 동작합니다.
